What is Cloud Gaming?

The concept of cloud gaming is a relatively new technology in which the game engine processing happens on a powerful server and the video stream is sent to any computer using nothing other than the power of the internet. Cloud gaming is a streaming service that provides high-quality gaming content but has high demands of internet bandwidth and server capacity.
